The type of file item. This can have one of the following values: - "delete folder" - a delete folder (doesn't require a source) - "delete" - a delete file (doesn't require a source) - "dll" - a dynamic link library - "exe" - an execuatable file - "folder" - a folder (doesn't require a source) - "folder all" - a folder and recursively all its files and subfolders - "normal" (default) - a normal file - "self-registering dll" - a self-registering dynamic link library - "text" - a text file - "tokenised" - a tokenised text file
